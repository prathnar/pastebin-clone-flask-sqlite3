# pastebin-clone-flask-sqlite3
### A Simple and Fast Way to Share Code Snippets and Text Online

# Overview

CodeDrop is a lightweight and user-friendly platform that allows users to share code snippets or text online quickly and securely.
Inspired by the concept of Pastebin, this project focuses on providing a clean, modern interface for developers, students, and professionals to create, share, and manage snippets easily.
The platform offers essential features such as syntax highlighting, live preview, paste expiration, and code organization, making it ideal for quick collaborations or personal storage of code snippets.
Through this project, the aim is to simplify code sharing and collaboration with a minimalistic design and smooth user experience.

# Features

### 1.Create and Share Snippets
#### Add and share any code or text instantly.
#### Generate unique URLs for each snippet.
### 2.Syntax Highlighting
#### Supports multiple programming languages for better readability.
### 3.Paste Expiration
#### Set a custom expiration time to automatically remove old pastes.
### 4.Live Preview
#### Instantly preview formatted code before sharing.
